
# Fake Artist

A fast-paced real-time multiplayer drawing and deduction game.

## Prereqs

1. node
2. yarn
3. git

## Installation

1. `git clone https://gitlabci.exxeta.com/hsreutlingen/ss19-team-2.git`

1. `cd ss19-team-2`

1. `yarn`

### Start Server

`nodemon server/server.js`

### Start Client

`yarn start`
