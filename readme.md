
# Fake Artist

A fast-paced real-time multiplayer drawing and deduction game.

## Prerequisites

1. node
2. yarn
3. git

## Installation

1. `git clone https://gitlabci.exxeta.com/hsreutlingen/ss19-team-2.git`

2. `cd ss19-team-2`

3. `cd fake-artist`

4. `yarn`

## Start Server

In the `fake-artist` directory, do:  
`yarn start-server`

## Start Client

In the `fake-artist` directory, do:  
`yarn start`
